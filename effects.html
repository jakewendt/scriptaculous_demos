This is a triple layer'd div. The outer layer is there as a spaceholder for when the second layer is not. The second layer is the layer on which all* effects are directed. The third layer holds all of the content. All 3 layers are fixed at 500px by 500px. Using this techinique seems to provide the best results.
(* - list effects are directed at a "ul" inside the third layer; gradient and pixelate create a new element that is appended to the target's parent (outer layer))
